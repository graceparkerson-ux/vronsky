# Vronsky

A reading app for people who want their reading life to feel like a conversation, not a database.

Live at **[vronsky.app](https://vronsky.app)**.

## What it is

Track what you're reading. Write reviews when you finish. Join threads about books, authors, and topics you love. Talk to other readers.

## What's here

This is the prototype — a single-page web app. Everything runs in the browser. Data is stored locally in your browser (so each visitor has their own reading history).

A real backend is coming next.

## Stack

- HTML, CSS, JavaScript (no framework yet)
- [Open Library API](https://openlibrary.org/dev/docs/api/search) for book search and covers
- Hosted on Vercel
